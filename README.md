# Solar System 3D

Интерактивная 3D-модель Солнечной системы на Three.js.

## Запуск

Открой `index.html` в браузере (нужен интернет для Three.js CDN).

Или локальный сервер:

```bash
npx --yes serve .
```

## Управление

**ПК:** WASD — полёт, мышь — осмотр, `1`–`8` — планеты, `H` — скрыть HUD, `F` — взлёт.

**Телефон:** джойстик, свайп, кнопки на экране.

## Структура

- `index.html` — игра
- `textures/` — карты планет и Солнца
