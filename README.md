# Solar Nemesis

Интерактивная 3D Солнечная система (Three.js) — полёт из кабины корабля.

**Онлайн:** [https://gunlinshaotan-dot.github.io/solar/](https://gunlinshaotan-dot.github.io/solar/)

## Управление

### ПК
- **Мышь** — поворот корабля  
- **Alt + мышь** — осмотр кабины  
- **WASD** — тяга · **Q/E** — крен · **Пробел/Ctrl** — вверх/вниз · **X** — тормоз  
- **Shift** — варп · **F / Пробел** — взлёт с поверхности · **H** — скрыть HUD  

### Android / телефон
- Открой ссылку в Chrome → «Начать полёт» (полноэкран)  
- **Джойстик** — тяга · **свайп** — поворот  
- **👁** (зажми) — осмотр кабины · **⚡** — варп · **ВЗЛЁТ** — с поверхности  
- Можно добавить на домашний экран (установить приложение)

## Локальный запуск

Дважды кликни **`start.bat`** или:

```bash
node server.js
```

Открой [http://127.0.0.1:3000](http://127.0.0.1:3000)
